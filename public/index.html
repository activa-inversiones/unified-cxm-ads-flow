Acquisition Hub

En progreso
Canales que ya entran o deben entrar al hub.
  • Meta Ads: webhook leadgen, leadgen_update, formulario premium
  • Google Ads: staging de conversiones offline + gclid
  • TikTok Ads: Contact + SubmitForm vía Events API
  • WhatsApp Ads / Messenger: conexión controlada, no invasiva

Lead Intelligence Hub

Operativo parcial
Qué ya funciona y qué debemos separar como módulo.
  • Scoring por reglas: ventanas, proyecto, presupuesto
  • Scoring IA: clasificación VIP / Normal
  • Zoho CRM: inserción automática
  • Falta: playbooks de ventas, SLA, recontacto y closed-loop

Insights Hub

Pendiente
Meta operativa mínima para no improvisar.
  • Objetivo: 1000 autores únicos monitorizados
  • Métricas: mentions, reach, sentiment, competitor, keyword
  • Ingesta: X, Instagram, Facebook, YouTube, LinkedIn, scraping permitido
  • Falta: conectores reales + modelo de alertas

Plan de construcción sostenida

Sin perder nada ganado
Fase 1Ahora
Consolidar Meta Ads → scoring → Zoho y dejar el formulario premium como principal.
Fase 2Siguiente
Crear módulo separado de Google Ads conversions con su propio endpoint y dashboard de retorno.
Fase 3Siguiente
Crear módulo TikTok Ads con eventos mínimos y validación end-to-end.
Fase 4Después
Social listening como servicio separado, pero visible desde este mismo Command Center.
Fase 5Después
Competitive radar y keyword intelligence con snapshots diarios y alertas.

Recomendación: construir cada hub como módulo separado y mantener un solo Command Center arriba. Así no rompes el sistema al crecer y puedes desplegar en forma sostenida.
